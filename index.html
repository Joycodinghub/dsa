Array

An array is a foundational data structure used extensively in programming. It serves as a container for holding a group of elements, all of which are usually of the same data type. These elements are stored in consecutive memory locations, creating a linear arrangement. The unique feature of arrays is the ability to access each element based on its index, which signifies its precise position within the array. In an array, the index starts from zero for the first element, one for the second, and so on. This indexing mechanism allows for efficient and rapid retrieval of specific elements. Arrays are especially valuable when dealing with substantial amounts of data, as they provide a structured approach to organizing and managing information. By utilizing arrays, programmers can enhance memory utilization and optimize the speed of operations like insertion, deletion, and retrieval of data. Arrays play a pivotal role in various algorithms and problem-solving approaches, acting as the backbone for many data manipulation tasks in programming.
Sorting
Sorting is arranging a group of data in a particular manner according to the requirement. The algorithms which help in performing this function are called sorting algorithms. Generally sorting algorithms are used to sort groups of data in an increasing or decreasing manner.

  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Merge Sort
  • Quick Sort
  • Cycle Sort
Searching
Searching algorithms are the ones that are used for searching elements or groups of elements from a particular data structure. They can be of different types based on their approach or the data structure in which the element should be found.

  • Linear Search
  • Binary Search
Rotate & Reverse
Rotating an array involves shifting its elements by a certain number of positions either to the left or to the right. The shifted elements wrap around the array, creating a circular effect. There are two common types of rotations: right rotation and left rotation.

  • Rotate
  • Reverse
Kadane's Algorithm
This algorithm is a variation of the maximum subarray problem that finds the maximum sum of a contiguous subarray within a one-dimensional array of numbers which has at least one positive number

  • Kadane's Algorithm
Sliding Window
This algorithm is used to solve problems that involve arrays or lists and a subarray or sublist. It involves creating a window of fixed size and sliding it over the array or list to find the desired subarray or sublist

  • Sliding Window
Dutch Flag algorithm
The Dutch National Flag algorithm is particularly efficient because it sorts the array in a single pass without using any sorting algorithms like quicksort or mergesort. It works by maintaining three pointers: one for the low end of 0s, one for the high end of 2s, and one to traverse the array.

  • Dutch Flag

String

A string is a sequence of characters used to represent text. It's a fundamental data type and an instance of the String class. Strings are enclosed in double quotes ("") and can contain letters, numbers, symbols, and whitespace. Java provides various methods to manipulate and compare strings. Since strings are immutable, any operation that seems to modify a string actually creates a new string object. String manipulation is common in programming for tasks like input/output, text processing, and data formatting. Java's String class offers powerful features for working with textual data, making it an essential component of Java programming.
String Searching
These algorithms aim to find one or several strings (patterns) within a larger string or text. They are also known as string-matching algorithms.The choice of algorithm depends on factors like the size of the text, pattern, and the specific use case. Each algorithm has its advantages and drawbacks, making it important to select the most appropriate one for the task at hand.

  • Rabin-Karp Algorithm
  • Knuth-Morris-Pratt Algorithm
String Matching
String matching in Java refers to the process of finding occurrences of a specific pattern within a larger text. Java provides various algorithms like the Knuth-Morris-Pratt (KMP) algorithm, Boyer-Moore algorithm, and more to efficiently locate and extract patterns from strings. These algorithms optimize character comparisons, utilize hashing, or create specialized data structures for faster searching.

  • Rabin-Karp Algorithm
  • Knuth-Morris-Pratt Algorithm
